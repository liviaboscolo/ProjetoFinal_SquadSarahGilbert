```git clone https://github.com/liviaboscolo/ProjetoFinal_SquadSarahGilbert.git```

instalar dependencias do projeto:<br>
```poetry install``` <br>

instalar um pacote <br>
```poetry add django==4.2.5```<br>

```poetry run python manage.py runserver```<br>
